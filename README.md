# Sistema de Generación Óptima de Horarios Académicos

Este proyecto es el resultado del **Proyecto de Fin de Asignatura (PFA)** para el curso Taller de Proyectos 2 de la Universidad Continental. El sistema utiliza el stack **MERN** para resolver el problema complejo de la planificación académica en entornos de currículo flexible.
